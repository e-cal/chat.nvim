# chat.nvim - Simple AI Chat

Chat with (any) LLM in neovim. No half baked bells and whistles, just chat with
the service of your choice and easy copy-pasting back and forth.

Editable chat history, popup window so no annoying last buffer issues,

## Goal / Credit

Combining the

- simplicity of [gp.nvim](https://github.com/Robitx/gp.nvim)
- flexibility of [ogpt.nvim](https://github.com/huynle/ogpt.nvim)
- nice UI of [neoai.nvim](https://github.com/Bryley/neoai.nvim)
